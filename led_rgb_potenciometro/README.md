## LED RGB com Potenciômetro

- controle cada uma das cores(R,G,B) com um potenciômetro.

- potenciômetro para R (0, 255) porta: A2
- potenciômetro para G (0, 255) porta: A1
- potenciômetro para B (0,255) porta: A0
